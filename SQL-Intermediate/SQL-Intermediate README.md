# 7. JOINs

Contains SQL queries practiced in MySQL to understand how different types of joins combine data across tables in the `parks_and_recreation` database.

## Covered Concepts
- Understanding `INNER JOIN` and default `JOIN` behavior
- Resolving column ambiguity using table-qualified column names
- Using table aliases for readability
- Selecting specific columns from joined tables
- `LEFT JOIN` and `RIGHT JOIN` (outer joins)
- Self joins using the same table with aliases
- Applying self joins for row-to-row relationships (e.g., Secret Santa logic)
- Joining more than two tables in a single query

## Database
- MySQL
- Schema: `parks_and_recreation`
- Tables:
  - `employee_demographics`
  - `employee_salary`
  - `parks_departments`
